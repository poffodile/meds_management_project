## The repeatable recipe (proven on Stock)
For each page, one step at a time:
1. **Controller** — add an `indexReact()` that returns `Inertia::render('Medication/<Page>', $props)` with data shaped into plain arrays. Reuse the legacy controller's queries.
2. **Route** — add `/medication/<page>-react` (+ any action routes), isolated. Legacy route untouched.
3. **Page** — `resources/js/Pages/Medication/<Page>.jsx`, built from shared components (`PageHeader`, `DataTable`, `StatusBadge`, `StatCard`, `FormModal`, `ConfirmDialog`). Wrapped in `AppShell`.
4. **Actions** — forms/feature bits in `frontend/features/medications/`.
5. **Roles** — managers get the actions; carers read-only where appropriate (enforced server-side too).
6. **Tests** — one backend feature test (auth + Inertia props) + one frontend render test. `php artisan test` / `npm run test` green.

## Recommended order
Simplest-to-hardest, to keep momentum: **Controlled Drugs → Missed Doses → Medication Round → Shift Handover.**
